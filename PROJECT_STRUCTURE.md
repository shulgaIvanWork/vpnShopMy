# MAX VPN Bot - Структура проекта

## 📁 Файлы верхнего уровня

### ⚙️ Конфигурация
- **`.env`** - Конфигурация проекта (токены, БД, серверы). **НЕ коммитить!**
- **`.env.example`** - Пример конфигурации для копирования
- **`package.json`** - Зависимости Node.js и скрипты запуска

### 🐳 Docker
- **`Dockerfile`** - Образ для контейнеризации бота (Node.js 20, multi-stage build)
- **`docker-compose.yml`** - Оркестрация: PostgreSQL + VPN Bot
- **`.dockerignore`** - Исключения для Docker build

### 📝 Документация
- **`README.md`** - Основная документация проекта
- **`DEPLOY.md`** - Пошаговая инструкция деплоя на сервер
- **`PROJECT_STRUCTURE.md`** - Этот файл (описание структуры)

### 🚀 Деплой
- **`deploy.sh`** - Bash скрипт для деплоя на Linux серверах

### 🗄️ База данных
- **`init.sql`** - SQL скрипт инициализации БД (таблицы, индексы)

### 🎬 Медиа
- **`videos/`** - Видео-инструкции для пользователей
  - `v2ray_instruction.mp4` - Инструкция для V2Ray
  - `hiddify_instruction.mp4` - Инструкция для Hiddify
  - `onexray_instruction.mp4` - Инструкция для OneXray

### 🔧 Исходный код
- **`index.js`** - Точка входа (запуск бота, миграции, cron)
- **`worker.js`** - Воркер для фоновых задач (если используется)
- **`src/`** - Основной исходный код
  - `bot/` - Обработчики сообщений, клавиатуры, middleware
  - `db/` - Модели БД, connection pool, миграции
  - `services/` - Бизнес-логика (XUI, AI, Cron, Referral)
  - `config.js` - Загрузка конфигурации из .env

## ✅ Все файлы нужны

Все файлы в корне используются:
- Docker файлы для контейнеризации
- SQL для инициализации БД
- Видео для инструкций
- Скрипты для деплоя
