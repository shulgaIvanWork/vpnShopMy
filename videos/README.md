# Video Instructions

Поместите сюда видеоинструкции для следующих приложений:

1. `v2ray_instruction.mp4` - Инструкция для v2rayN (Windows) / v2rayU (macOS)
2. `onexray_instruction.mp4` - Инструкция для oneXray (macOS/iOS)
3. `hiddify_instruction.mp4` - Инструкция для Hiddify (Android/iOS)

**Формат видео:**
- MP4 (рекомендуется)
- MOV
- AVI

**Размер:**
- Старайтесь сжимать видео до 10-20 МБ
- Оптимальная длительность: 1-3 минуты

**Содержание видео:**
1. Установка приложения
2. Добавление VPN подключения из ссылки
3. Подключение к серверу
4. Проверка работы VPN
